## Featured projects

### [Lasso](https://github.com/thomasfevre/lasso)

**Visual context for AI coding agents on macOS.**

Capture a deliberate rectangle, add pins or notes, and let Claude Code, Codex, or Cursor retrieve the visual and structured context through MCP. Lasso is local-first, open source, lightweight, and distributed as a notarized macOS app.
